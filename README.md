# PESUnity
PESUnity is a comprehensive web platform enabling PES University students to share academic resources, collaborate on projects, and engage in real-time discussions through an intuitive message board system

## project files

### you can find all files related to the project in the master branch of the same repository. this includes all essential components, from the frontend and backend to the database configurations and additional resources required to run the platform smoothly.
